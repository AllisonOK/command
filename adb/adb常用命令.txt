adb wifi调试
adb tcpip 5555
adb connect 手机ip（如adb connect 192.168.1.101）
adb devices验证是否连接成功
给指定的手机安装程序
adb -s 手机编码 install xx.apk

添加DeviceOwner
adb shell dpm set-device-owner com.afwsamples.testdpc/.DeviceAdminReceiver
adb shell dpm set-device-owner com.easycontrol.emm/.receiver.DeviceOwnerReceiver
com.abupdate.appcenter

移除DeviceOwner
adb shell dpm remove-active-admin com.afwsamples.testdpc/.DeviceAdminReceiver
adb shell dpm remove-active-admin com.easycontrol.emm/.receiver.DeviceOwnerReceiver
com.easycontrol.emm

查看所有包名
adb shell pm list packages

启动
adb shell am start -n 包名/MainActivity名

adb shell pm clear com.abupdate.appcenter

adb uninstall jp.smartmobile.quickdischarge

截取fb0界面到sdcard，并导入到电脑，可以进行查看。
adb shell screencap -p sdcard/fb0.png

adb pull /sdcard/fb0.png

adb shell rm /sdcard/fb0.png

截取fb1界面到sdcard，文件名为fb1.png
adb shell screencap -d /dev/graphics/fb1 sdcard/fb1.png

执行命令，抓取日志
adb logcat -v time >.\\logcat.log      开始抓取日志，并保存到当前路径下，文件名为 logcat.log

Ctrl+c    结束日志抓取

adb logcat -c 清空旧日志

adb logcat -s "tag" adb打印指定tag的logcat

adb logcat | findstr tag名称

复制文件
copy  E:\music\*陈奕迅*.*  E:\test

查看系统Linux kernel版本号
cat /proc/version

获取build number
adb shell getprop ro.build.display.id

监听手机打开的页面所属包名
adb shell am monitor

adb命令查看应用权限赋予情况
adb shell dumpsys package com.test.abc

应用管理
	查看应用列表
		adb shell pm list packages  [-f] [-d] [-e] [-s] [-3] [-i] [-u] [--user USER_ID] [FILTER]
		
如何查看任务栈		
adb shell dumpsys activity -p com.packagename.packagename > taskinfos.txt
搜索关键字：Hist、TaskId
